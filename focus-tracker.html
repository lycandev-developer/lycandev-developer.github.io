FocusTracker

Platform: Windows/Linux

Description: FocusTracker is a lightweight, offline tool that silently logs window focus events so you can see where your time is actually going. Great for developers, freelancers, or anyone managing time without the cloud.

Features:

  • Tracks app usage without internet
  • Lightweight and privacy-respecting
  • Outputs daily CSV logs
  • CLI + optional GUI
